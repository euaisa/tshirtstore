Description of the course project

The idea of our project is pretty intuitive if you navigate throughout the website. We have a home page with the description of our project and a button that takes you to the "Products" page. In the Products page, users can view products (t-shirts) and add them to the 'cart' which will be displayed in the "Cart" page. Another functionality of the "Cart" page is that users can add reviews/rate/comment the products and these will be displayed in the "Reviews" page. All pages have a header, content and footer section.
